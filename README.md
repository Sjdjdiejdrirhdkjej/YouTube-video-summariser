# Video Summarizer App

React + TS + Vite app to summarize YouTube videos (demo UI with theme toggle).

## Setup

```bash
npm install
npm run dev
```

## Features

- Enter YouTube URL → extracts ID → demo summary
- Dark/light theme
- Responsive UI

## Production

Add backend for transcripts (YouTube API) + AI summary (OpenAI/etc.).

Copy .env.example → .env with keys.
